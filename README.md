# Madhav Ecommerce Sales Dashboard

## Project Title

**Madhav Ecommerce Sales Dashboard**

## Short Description / Purpose

An interactive Power BI dashboard designed to analyze and visualize ecommerce sales data. The project helps track key business metrics such as sales, profit, quantity sold, customer behavior, payment preferences, and regional performance, enabling data-driven decision-making.

## Tech Stack

* Power BI Desktop
* Power Query
* DAX (Data Analysis Expressions)
* Data Modeling
* Data Visualization

## Data Source

The dataset contains ecommerce transaction records, including customer details, product categories, sub-categories, payment modes, order quantities, sales amounts, profits, cities, and order dates.

## Features / Highlights

* KPI Cards displaying Total Sales, Profit, and Quantity Sold.
* Category-wise analysis for Clothing, Electronics, and Furniture.
* Customer-wise sales performance tracking.
* Sub-category profit analysis using donut charts.
* Payment mode distribution (COD, Credit Card, Debit Card, EMI, UPI).
* Monthly and quarterly sales/profit trend analysis.
* Interactive map showing city-wise sales quantity.
* Detailed transaction table for granular data exploration.
* Dynamic slicers and filters for interactive reporting.
* Clean and user-friendly dashboard layout for business insights.

This project demonstrates the use of Power BI to transform raw ecommerce data into meaningful visualizations and actionable business intelligence.
